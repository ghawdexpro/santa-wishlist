## Project: Santa's Nice List - Christmas Wishlist App
A full-stack app where users can:
1. Create wishlists for Christmas
2. Share lists with family/friends
3. Mark gifts as "claimed" (secret from recipient)
4. Track Santa's journey on Christmas Eve

## Tech Stack
- Frontend: Next.js 14 + Tailwind CSS
- Backend: Supabase (Auth + Database + Realtime)
- Deployment: Railway
- Festive: Snow animations, Christmas theme
